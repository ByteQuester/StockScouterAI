class app.services.service_manager.DataPipelineIntegration(cik_number: str = None, use_snowflake: bool = True, snowflake_config: SnowflakeConfig = None, local_storage_dir: str = 'data')[source]#

Bases: object

Integrates various data processing stages for SEC data, including fetching, preprocessing, processing, and transforming into JSON format.

cik_number#

Central Index Key (CIK) number for querying SEC data.

Type:

str

use_snowflake#

Flag to determine if Snowflake database is used for storage.

Type:

bool

fetch_data()[source]#

Fetches data from the SEC API using a CIK number.

preprocess_data()[source]#

Preprocesses raw data fetched from the SEC API.

process_and_store_data()[source]#

Processes preprocessed data and stores or uploads results.

transform_and_store_json()[source]#

Transforms data into JSON format and stores it.

Example

>>> cik_number = '0000012927'
>>> use_snowflake = False
>>> data_pipeline = DataPipelineIntegration(cik_number, use_snowflake)
>>> raw_data = data_pipeline.fetch_data()
>>> preprocessed_data = data_pipeline.preprocess_data(raw_data)
>>> processed_data = data_pipeline.process_and_store_data()
>>> json_data = data_pipeline.transform_and_store_json()
